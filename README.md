# 📌 Quantity Measurement System

This project is a Java application that shows how to measure and compare different units like Feet, Inch, Yard, and Centimeter.

It is built step-by-step using multiple use cases (UC1 to UC8).

---

## 🎯 What this project does

- Compare measurements (Feet, Inch, Yard, CM)
- Convert one unit to another
- Add two measurements
- Follow clean coding practices
- Build a simple and scalable design

---

## 🧱 Project Structure

- **main** → Contains README (documentation only)
- **dev** → Contains full working code
- **feature branches** → Each use case (UC1–UC8)

---

## 🔄 Workflow

- Each feature is developed in a separate branch
- Code is tested
- Then merged into **dev**
- **main** is kept clean

---

## 💡 Concepts Used

- Java
- OOP (Object-Oriented Programming)
- Enum for units
- DRY principle
- Unit conversion logic
- Git branching

---

## 🚀 Features

- Check measurement equality
- Convert units
- Add measurements
- Choose result unit
- Easy to extend
